# Explorer11

A Windows 11-style file explorer for Windows 10.

## Features

- Modern Windows 11 UI design
- File browsing and navigation
- Quick access to common folders (Home, Documents, Downloads, Pictures)
- Folder icons with emoji

## How to Use

1. Double-click `Explorer11.exe`
2. Navigate your files

## Building

Requires .NET 6.0 SDK installed.

```bash
dotnet publish -c Release -r win-x64 --self-contained
```
